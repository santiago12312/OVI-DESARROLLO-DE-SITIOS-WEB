DESARROLLO DE SOFTWARE

OBJETIVOS GENERALES

Dar a entender al usuario la facilidad con la que se inicia el desarrollo de un software

Objetivo específico 1:

Tener la facilidad de iniciar con el desarrollo de cualquier software extrayendo los requisitos necesarios para poder dar inicio a la creación, para luego entrar en el diseño y la arquitectura de los que va a tratar nuestro software.

Objetivo específico 2:

El usuario va a entender cómo realizar la programación del software para luego entrar a realizar las pruebas de funcionamiento.

Objetivo específico 3:

El usuario entenderá como realizar la documentación y el mantenimiento sobre los diferentes inconvenientes que tuvo al realizar el software, así también realizara un manual explicando cómo funciona dicho programa.
